Viedo tags and its Attribute

1. Controls:- With the help of this attribute You can control the viedo(pause/play, download e.t.c.)
2. Loop:- With the help of this attribute You can reapeat the viedo.
3. Mutted:- With the help of this attribute You can mute and unmute the playing viedo.
4. Poster:- With the help of this attribute You can add a poster(like YouTube thubnail) to your viedo.
5. Autoplay:- With the help of this attribute you can autoplay your viedo.

Audio tags and its Attribute

1. Controls:- With the help of this attribute You can control the audio(pause/play, download e.t.c.)
2. Loop:- With the help of this attribute You can reapeat the audio.
3. Mutted:- With the help of this attribute You can mute and unmute the playing audio.
4. Autoplay:- With the help of this attribute you can autoplay your audio.
5. Preload:- Specifies if and how the audio should loaded when the page loads('auto', 'metadata', 'none').
